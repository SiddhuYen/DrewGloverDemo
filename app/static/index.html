Network intelligence

Artemis

warm-intro router

Every hop is backed by a source that asserts it. No path is invented, and it says so plainly when none exists.

01
Export your connections from LinkedIn.

Go to Settings & Privacy → Data privacy → Get a copy of your data. Tick Connections — not the full archive — and request it. LinkedIn emails you a Connections.csv, usually within minutes.

02
Drop that file here.

It is read on this machine and never uploaded anywhere. Each row becomes a tier 1 connection — the warmest kind, because you are asserting it yourself rather than inferring it from a page.

or drag it onto this box
Why this is worth doing. The built-in network is what public sources assert: episodes, rosters, filings. Your own first degree is invisible to all of them. Import it and every one of those people becomes a real starting point, so paths that were four cold hops can collapse to one warm introduction you can actually make.
01
Paste a Brave Search API key.

Free at brave.com/search/api — the free tier is 2,000 queries a month and needs no card. Without a key the app still works: it falls back to DuckDuckGo, which rate-limits hard and finds fewer roster pages.

02
Run a deep search.

An ordinary query gives up after 40 seconds so the window stays responsive. A deep search keeps reading — rosters, filings, episodes — for up to 4 minutes to find a route to someone the graph has never heard of. The app is busy for the whole time.

A search engine cannot create a connection. Brave only tells the app which page to read. The page still has to assert the relationship — a roster listing both people, an episode, a filing — or no edge is made. A better search engine finds more pages; it never lowers the bar for what counts.
What this shows. Every hop comes from a source that structurally asserts the relationship — a team roster, a podcast episode, an SEC filing, a Wikidata claim. Two names appearing on the same page is not a relationship, and never creates a hop. Paths are real but unverified: confirm before asking for the intro. Coverage is the VC/startup world, not the general public.
Warmth tiers — 1 demonstrated (podcast, co-founder, direct connection) · 2 same small firm / shared board · 3 investor–founder · 4 shared portfolio or employer · 5 weak affiliation. ★ marks the seed's real first degree.
